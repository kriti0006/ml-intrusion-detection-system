# Machine Learning-Based Intrusion Detection System

This project implements an Intrusion Detection System using Machine Learning. It classifies network traffic as Normal or Attack using the NSL-KDD dataset.

## Dataset
- NSL-KDD

## Algorithms Used
- Decision Tree Classifier
- Random Forest Classifier

## Preprocessing
- One-hot encoding for categorical features
- StandardScaler normalization for numerical features
- Label conversion into Normal and Attack classes

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- Feature Importance

## Tools and Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib
